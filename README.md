# 📊 Vrinda Sales Dashboard

## Project Overview

The **Vrinda Sales Dashboard** is an interactive Excel dashboard built to analyze sales performance, customer behavior, and order trends. The dashboard transforms raw sales data into meaningful insights using Pivot Tables, Pivot Charts, Slicers, and Excel formulas, enabling quick and data-driven decision-making.

## Objectives

* Analyze overall sales performance.
* Track monthly sales and order trends.
* Understand customer demographics.
* Identify top-performing states and sales channels.
* Evaluate order status and business performance.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Data Cleaning
* Excel Functions (SUMIFS, COUNTIFS, IF, TEXT, etc.)

## Key KPIs

* Total Sales
* Total Orders
* Sales by Month
* Sales by Gender
* Sales by Age Group
* Sales by State
* Sales by Sales Channel
* Order Status (Delivered, Cancelled, Returned, Refunded)

## Dashboard Insights

* Identified monthly sales trends and peak sales periods.
* Compared purchasing behavior across different age groups and genders.
* Highlighted the top revenue-generating states.
* Analyzed the contribution of various sales channels such as Amazon, Flipkart, Myntra, and Ajio.
* Monitored order fulfillment performance by tracking delivery, cancellation, return, and refund rates.
* Enabled interactive filtering using slicers for faster business analysis.

## Business Impact

This dashboard helps stakeholders:

* Monitor sales performance in real time.
* Identify profitable customer segments.
* Optimize marketing strategies based on customer behavior.
* Improve inventory planning using sales trends.
* Support data-driven business decisions.
